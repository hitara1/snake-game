# 🐍 Snake Game

A classic Snake game built with vanilla HTML, CSS, and JavaScript. Play it right in your browser — no installation needed!

> 🎮 [Play Now](https://hitara1.github.io/snake-game/) *(Enable GitHub Pages in Settings to host live)*

---

## ✨ Features

- **Smooth Gameplay** — Responsive controls with progressive speed increase
- **Particle Effects** — Satisfying visual feedback when eating food
- **Animated Snake Eyes** — The snake watches where it's going
- **Score Tracking** — Current score + persistent high score via localStorage
- **Progressive Difficulty** — Snake speeds up as your score grows
- **Dark Neon Theme** — Easy on the eyes with glowing accents
- **Mobile Support** — Touch-friendly D-pad controls for on-the-go play

---

## 🕹️ How to Play

1. Open `index.html` in any modern web browser
2. Click **"Start Game"** to begin
3. Guide the snake to eat the red food pellets
4. Avoid hitting the walls or yourself
5. Try to beat your high score!

---

## 🎮 Controls

| Action | Keys |
|--------|------|
| Move Up | `↑` Arrow or `W` |
| Move Down | `↓` Arrow or `S` |
| Move Left | `←` Arrow or `A` |
| Move Right | `→` Arrow or `D` |

**Mobile:** Use the on-screen D-pad buttons

---

## 🚀 Quick Start

### Play Locally
```bash
git clone https://github.com/hitara1/snake-game.git
cd snake-game
# Open index.html in your browser
```

### Host with GitHub Pages
1. Go to **Settings** → **Pages** in your repo
2. Set Source to `Deploy from a branch`
3. Select `main` branch and `/ (root)` folder
4. Save and your game will be live at `https://hitara1.github.io/snake-game/`
